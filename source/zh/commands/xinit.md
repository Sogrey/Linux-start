# xinit - Linux下X-Window系统的初始化程序


xinit命令 是Linux下X-Window系统的初始化程序，主要完成X服务器的初始化设置。

## 语法

``` bash
xinit  [OPTION]
```

## 参数
- 客户端选项：客户端指令及选项；
- --：用于区分客户端选项和服务器端选项；
- 服务器端选项：服务器端选项指令及选项。
