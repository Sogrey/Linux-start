# ntpstat - 用于显示本机上一次和服务器同步时间的情况

## 语法

``` bash
ntpstat
```

## 选项
无
## 举例

``` bash
[root@localhost ~]$ ntpstat
unsynchronised
  time server re-starting
   polling server every 8 s
```
