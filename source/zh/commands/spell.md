# spell - 对文件进行拼写检查

spell命令 对文件进行拼写检查，并把拼写错误的单词输出。

## 语法

``` bash
spell  file
```


## 举例

``` bash
$ cat test.txt
Holle world
$ spell test.txt
Holle # 检查出 Holle 拼写错误
```
