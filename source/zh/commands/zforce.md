# zforce - 强制gzip格式的文件使用gz后缀名

这样可以放置gzip被压缩两次。

## 语法

``` bash
zforce   gzip格式文件
```

## 选项

``` bash
-L                               # 从环境变量中使用PWD
-P                               # 跳过所有的符号链接
--help                           # 显示帮助文档
--version                        # 显示命令版本信息
```
## 举例

``` bash
[sogrey@bogon 文档]$ pwd
```
