# usernetctl - 被允许时操作指定的网络接口

usernetctl命令 在用于被允许时操作指定的网络接口。

## 语法

``` bash
usernetctl  [param]
```

## 选项

``` bash
网络接口     # 被操纵的网络接口；
up          # 激活网络接口；
down        # 禁用网络接口；
report      # 报告网络接口状态。
```
