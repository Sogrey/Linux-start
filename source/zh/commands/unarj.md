# unarj - 解压缩由arj命令创建的压缩包
unarj命令 用来解压缩由arj命令创建的压缩包。

## 语法

``` bash
unarj  [OPTION]  [参数]
```

## 选项

``` bash
e              # 解压缩.arj文件；
l              # 显示压缩文件内所包含的文件；
t              # 检查压缩文件是否正确；
x              # 解压缩时保留原有的路径。
```
